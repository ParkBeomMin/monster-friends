# 몬스터 프렌즈 (가제)

귀여운 픽셀 몬스터를 모아 잃어버린 지역을 되찾는 **방치형 오토배틀러**.

- 장르: 방치형(Idle) + 오토배틀러 + 비동기 PvP 아레나
- 감성: 메이플 느낌의 귀여운 픽셀 판타지
- 플랫폼: 웹(브라우저) 우선, 추후 모바일 래핑
- 목표: 1인 개발로 만들고 싶은 게임을 만들고, 광고로 용돈 수준 수익
